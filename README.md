## Reskeleton

<img src="./reskeleton-image.gif"/>

Easy and beautiful ReactReason skeleton component

# Install

- With npm

```bash
npm install --save reskeleton
```

- With yarn

```bash
yarn add reskeleton
```

Add `"reskeleton"` to `bsconfig`:

```js
"bs-dependencies": [
    ...
    "reskeleton"
],
```
