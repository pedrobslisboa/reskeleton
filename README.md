# Install

- With npm

```bash
npm install --save reskeleton
```

- With yarn

```bash
yarn add reskeleton
```

Add `"reskeleton"` to `bsconfig`:

```js
"bs-dependencies": [
    ...
    "reskeleton"
],
```
